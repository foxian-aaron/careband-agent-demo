# Demo Personas

## TEST001

Purpose:

- Display name: `團隊 Apple Watch 測試資料`.
- Team member Apple Watch / Apple Health test data.
- Used to demonstrate real wearable data import into DailySnapshot rows.
- Not real elderly user data.
- Not a medical record.
- Not a real elder health conclusion.

Use during presentation:

- Show `data_source = Apple Health Export`.
- Show `data_quality`.
- Show baseline status such as `baseline building` or `7-day baseline`.
- Explain that only daily aggregated data is used.

Do not say:

- "This is real elder data."
- "This proves a diagnosis."

## E001 Chen Bo

Purpose:

- Display name: `陳伯 Demo 情境`.
- Main scripted elderly care-loop demo.
- Shows activity decline, voice symptom, SOS, caregiver task, Agent summaries,
  and task completion.

Use during presentation:

- Trigger voice symptom.
- Trigger SOS.
- Show caregiver task creation.
- Show family/institution summaries.
- Complete the care task.
